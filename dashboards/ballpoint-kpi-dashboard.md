# Ballpoint KPI Dashboard

## Objective

Track Ballpoint performance using customer, revenue, and activity KPIs.

## Dataset

`datasets/village-market-recreation-data.csv`

## KPIs

- Daily Visitors
- Daily Revenue
- Average Revenue per Visitor
- Activity Usage
- Customer Type Mix

## Ballpoint-Specific Metrics

- Pool Usage
- Board Game Usage
- Walk-in Customers
- Member Customers

## Dashboard Visuals

- Revenue Trend
- Visitors Trend
- Activity Breakdown
- Customer Type Breakdown
- KPI Cards

## Business Questions

1. How much revenue does Ballpoint generate?
2. Which Ballpoint activity performs best?
3. What customer type visits Ballpoint most?
4. How can Ballpoint increase weekday revenue?

## Recommendations

- Promote board games during low-traffic periods.
- Create bundled offers for Pool + Board Games.
- Track repeat customers and member usage.
- Use KPI monitoring for daily reporting.
